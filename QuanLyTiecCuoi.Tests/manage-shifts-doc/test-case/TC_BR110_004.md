### Message Code

| Code  | Vietnamese                                    | English                           |
| ----- | --------------------------------------------- | --------------------------------- |
| MSG74 | Thời gian kết thúc phải sau thời gian bắt đầu | End time must be after start time |

### Validation Rule

```csharp
if (EndTime <= StartTime)
{
    ShowMessage("MSG74", "Thời gian kết thúc phải sau thời gian bắt đầu");
    return;
}
```

### Test Scenarios

| Scenario         | StartTime | EndTime | Valid? |

| End after Start  | 08:00     | 12:00   | ✓      |
