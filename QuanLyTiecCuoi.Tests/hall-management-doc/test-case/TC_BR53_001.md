### Service Method

```csharp
public async Task<bool> Delete(int hallId)
{
    var hall = await _context.Halls.FindAsync(hallId);
    if (hall == null) return false;

    _context.Halls.Remove(hall);
    await _context.SaveChangesAsync();
    return true;
}
```
