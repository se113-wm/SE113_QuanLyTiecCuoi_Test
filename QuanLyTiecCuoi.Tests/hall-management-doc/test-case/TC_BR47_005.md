### ID Preservation Logic

```csharp
var hallDto = new HallDTO
{
    HallId = SelectedItem.HallId,  // Preserve original ID
    HallName = HallName,
    HallTypeId = SelectedHallType.HallTypeId,
    MaxTableCount = MaxTableCount,
    Note = Note
};
await _hallService.Update(hallDto);
```
