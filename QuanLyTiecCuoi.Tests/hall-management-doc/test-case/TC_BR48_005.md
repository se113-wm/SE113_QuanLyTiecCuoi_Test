### Validation Logic

```csharp
// ExistsByName excludes current hall's ID
public bool ExistsByName(string hallName, int excludeHallId)
{
    return _context.Halls.Any(h =>
        h.HallName == hallName && h.HallId != excludeHallId);
}
```
